Dell Technologies Rewards Landing Page

Dell Rewards needed a high-converting landing page that clearly communicated program benefits and drove sign-ups. I designed a user-first experience that kept the brand's visual integrity while simplifying the enrollment decision.

+18% increase in Enrollment
-10% decrease in Bounce Rate
+50% conversion lift (12% to 18%)
Objective

Translate a feature-rich rewards program into a simple, compelling story while making enrollment the clearest next action.

Role

UI Designer
Landing page redesign

Industry

Technology
Personal PC

Timeframe

3 weeks

Overview

Dell Rewards needed a landing page that communicated value quickly and made the enrollment path feel simple. The project focused on balancing brand trust with clearer messaging, stronger hierarchy, and a more guided decision flow for users evaluating whether to join.

Problem

The challenge was translating a rewards program with many features into a page users could scan fast. The old structure carried useful content, but it needed tighter hierarchy and a clearer path toward sign-up.

Who it's for

  • Users comparing membership and rewards value quickly
  • Shoppers deciding whether to enroll before purchasing
  • Visitors who need confidence in how points and benefits work
  • Users on desktop and mobile who expect a frictionless sign-up path

What needed work

  • Program details needed to be easier to scan in sequence
  • CTA visibility needed to be stronger across the page
  • Benefits needed clearer grouping and pacing
  • Content needed to reduce hesitation before enrollment
  • The page needed to preserve brand trust while simplifying decisions

Page Flow

The page was structured into clear, goal-focused sections that guide users from understanding to action.

Flow Breakdown

  • How Dell Rewards Works: Step-by-step clarity on joining, earning, and redeeming.
  • Earning Points: Visual explanation of how purchases build value.
  • Exclusive Offers: Benefits framed as member-only advantages.
  • FAQ: Objection handling to reduce enrollment hesitation.

Design Approach

The layout followed a direct, benefit-led structure with clear hierarchy. The headline focused on immediacy and value while the visual system stayed consistent with Dell's design language.

The redesign used Dell's signature blue palette, neutral supporting tones, and modular content blocks to keep the experience branded, trustworthy, and easy to navigate.

Execution

Strategic CTA placement and cleaner module sequencing kept enrollment top-of-mind without making the experience feel aggressive.

What Changed

  • Simplified the rewards story into clearer, scannable chunks
  • Prioritized enrollment action with stronger CTA visibility
  • Aligned each section to a single user question
  • Used layout and spacing to reduce cognitive load
  • Maintained Dell brand consistency across all touchpoints
  • Kept modules responsive for desktop and mobile parity
  • Balanced information density with visual breathing room
  • Reinforced trust through structure and visual consistency

Dell Rewards Performance Improvements

The redesigned landing page improved enrollment intent by making program value easier to understand and reducing friction between sections. Clearer hierarchy and stronger CTA emphasis helped users stay engaged and move toward sign-up.

Learnings

This project showed how simplifying hierarchy can unlock performance even when the content set is complex. The biggest gains came from structuring information in a clearer sequence, preserving trust through brand consistency, and keeping the enrollment path visible throughout the experience.

Aligning messaging, layout, and interaction intent produced stronger engagement and conversion outcomes without overcomplicating the page design.
